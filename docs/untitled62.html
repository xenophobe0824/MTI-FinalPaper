XSL-FO应用于DITA文档

简要介绍在Oxygen XML Editor中发布PDF时应用样式表的方法。

Oxygen XML Editor是集成DITA的写作和发布于一体的商业软件，内置DITAOT, 可直接通过图形界面操作进行调用。

在 Oxygen XML Editor发布 PDF时，可以通过指定 Customization文件夹下的一些配置文件来对默认样式进行覆写，从而达到自定义样式的目的。

准备工作：

  1. 按照下图所示路径，在本地文件中找到 Customization文件夹：
    • {$Oxygen安装目录} \ frameworks \ dita \ DITA-OT3.x \ plugins \ org.dita.pdf2\
    • 注：{$Oxygen 安装目录} 指 Oxygen XML Editor 在当前计算机系统中的安装位置。
  2. 复制该文件夹到任意其他位置（建议复制到方便浏览的位置）。
  3. 在复制后的 Customization 文件夹中，去掉每个文件的 .orig 后缀（包括 fo 文件夹下的 attrs \ custom.xsl.orig 以及 xsl \ custom.xsl.orig）：
  4. 引入 custom.xsl 文件：打开 Customization 文件夹下的 catalog.xml 文件，将图中圈出的两行代码取消注释：

将样式表应用到Oxygen XML Editor中的Dita文档：

  1. 在Oxygen XML Editor中选择目标 ditamap 文件并打开。
  2. 点击工具栏中的配置按钮（Configure Transformation Senario(s)）：
  3. 选择DITA Map PDF并复制（duplicate）：
    在其他更高版本的Oxygen XML Editor中，DITA Map PDF或显示为DITA Map PDF - based on XSL-FO
  4. 在 parameters 中选择 customization.dir 参数并双击下图所示位置以设置 Customization 文件夹 路径（之前选择保存的路径）：
  5. 配置完成后选择 Apply associated
