GitHub Desktop下载与安装方法

简要介绍GitHub Desktop的下载与安装方法。

您可以在任何支持的操作系统上安装 GitHub Desktop。 有关详细信息，请参阅“支持的操作系统”。

  1. 要安装 GitHub Desktop，请访问 GitHub Desktop 的下载页面。 有关详细信息，请参阅“安装 GitHub Desktop”。
  2. 在安装 GitHub Desktop 后，您可以使用您在 GitHub 或 GitHub Enterprise 上的帐户验证应用程序。 身份验证允许您连接到 GitHub 或 GitHub Enterprise 上的远程仓库。
    1. 必须有帐户才可向 GitHub 或 GitHub Enterprise 验证。 有关创建帐户的更多信息，请参阅“注册新 GitHub 帐户”或联系 GitHub Enterprise 站点管理员。
    2. 在“文件”下拉菜单中，单击“选项”。 在选项窗口中，单击“帐户”并按照步骤登录。 有关详细信息，请参阅“向 GitHub 进行身份验证”。
