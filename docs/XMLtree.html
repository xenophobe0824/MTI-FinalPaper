XML文档的树状结构

XML 文档形成了一种树结构，它从"根部"开始，然后扩展到"枝叶"。

仍以“什么是XML”的XML文档作为演示的实例，此处介绍XML的树状结构。

Table 1. XML文档结构
XML语句 种类 Category
<?xml version="1.0” encoding="UTF-8"?> XML声明 /
<concept> 根元素 <root>
<title> 子元素 <child>
<shortdesc> 子元素 <child>
<conbpdy> 子元素 <child>
<p> 次级子元素 <subchild>

XML 文档必须包含根元素。该元素是所有其他元素的父元素。

XML 文档中的元素形成了一棵文档树。这棵树从根部开始，并扩展到树的最底端。

所有的元素都可以有子元素：

<root>

<child>

<subchild>.....</subchild>

</child>

</root>
